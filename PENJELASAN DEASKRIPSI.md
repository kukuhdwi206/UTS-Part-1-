# Tugas UTS Part 1 - Struktur Data (Sorting Mahasiswa)

Repository ini berisi implementasi algoritma pengurutan (Sorting) menggunakan bahasa pemrograman Java untuk memenuhi tugas UTS Part 1. Program ini mengurutkan data mahasiswa berdasarkan nilai secara **Descending** (dari nilai tertinggi ke terendah).

## Deskripsi Tugas
Tugas ini mencakup implementasi tiga teknik pengurutan utama, yaitu:
1. **Selection Sort**: Mencari nilai maksimal dan menempatkannya di posisi awal.
2. **Bubble Sort**: Membandingkan elemen yang bersebelahan dan menukarnya jika perlu.
3. **Insertion Sort**: Menyisipkan elemen ke posisi yang tepat dalam daftar yang sudah terurut.

Setiap program menampilkan **Proses Literasi** (langkah demi langkah) sehingga alur algoritma dapat dipahami dengan jelas melalui output terminal.
